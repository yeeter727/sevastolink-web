# sevastolink-web
A few modifications to [this existing HTML/CSS/JS project](https://github.com/ahmedmunzir/Sevastolink-Terminal) (by **ahmedmunzir**) designed to look like the Sevastolink terminal from Alien: Isolation. \
List of modifications made:
- Sevastolink logo custom GIF (animated by me)
- show boot screen on each refresh
- display the `/terminal/` page as the ending URL instead of `/postload.html`
- 2 entry system for each folder, as used in the game
- made it easier to customize the content by using `config.js`
- loading bar moves in uneven chunks
